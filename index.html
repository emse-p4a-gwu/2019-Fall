3 Course Description

3.1 GW Bulletin Description

Introduction to programming for data analytics using the R computing language; topics include R, RMarkdown, functions, conditionals, loops, debugging, testing, strings, file input/output, algorithmic thinking, coding style, efficiency, information visualization, Monte Carlo methods, webscraping data, packages, .

3.2 Detailed Description

Introduction to programming for data analytics using the R programming language. Topics covered include introduction to the R programming language, structured program design in R, data structures for analytics in R, information visualization, file input/output, basic data analytics methods, data acquisition from real world sources. This course will prepare students for higher level courses in data analytics.

This course provides undergraduate students with a foundation in exploring data in the R programming language. Students will learn how to work with a wide variety of data types and structures as well as fundamental concepts in information visualization. The course will be managed on Github, and students will learn the basics of using git and Github to manage version controlling. All analyses will be conducted to support reproducibility from raw data to results using RMarkdown. There are no prerequisites, and no prior programming experience is necessary to succeed in this class.

4 Learning objectives

Having successfully completed this course, students will be able to:

  • Conduct a systematic exploratory data analysis (EDA) of different types of data.
  • Understand fundamental principles of visualizing information for exploratory analysis and communication.
  • Organize projects and manage version controlling with Git and GitHub
  • Wrangle data from its original format into a fit-for-purpose format.
  • Get data off the web and expose data, code, results on the web.
  • Generate fully reproducible reports that contain contain code, equations, visualizations, and narrative text.
